## 二分法

只要看到排序数组查找的，就可以使用二分法(分治思想)

 统计一个数字在排序数组中出现的次数。 

```
输入: nums = [5,7,7,8,8,10], target = 8
输出: 2
```

```javascript
var search = function (nums, target) {
    //二分法
    let left = 0;
    let right = nums.length - 1;
    while (left < right) {
        let mid = (left + right) >>> 1  //相当于除以2
        if (nums[mid] < target) {
            left = mid + 1;
        } else {
            right = mid;
        }
    }
    let count = 0;
    while (left < nums.length && nums[left] === target) {
        left++;
        count++;
    }
    return count;
};
```
